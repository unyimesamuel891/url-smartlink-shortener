A **production-ready full-stack URL shortener** with:

- ✅ Minimal Brutalism aesthetic
- ✅ Complete documentation

## 🎯 Your Path Forward

Choose where you want to start:

### 1️⃣ **Get It Running Locally** (5 minutes)

Start here if you want to see the app working on your machine.

```bash
pnpm install
pnpm dev
```

Then open [http://localhost:3000](http://localhost:3000)

**Guide:** [QUICKSTART.md](./QUICKSTART.md)

### 2️⃣ **Push to GitHub** (10 minutes)

Share your code on GitHub for recruiters and collaborators to see.

```bash
git init
git add .
git commit -m "Initial commit: Smart Link Shortener"
git remote add origin https://github.com/YOUR_USERNAME/smart-link-shortener.git
git push -u origin main
```

**Guide:** [GITHUB_SETUP.md](./GITHUB_SETUP.md)

### 3️⃣ **Deploy to Production** (15 minutes)

Make your app live on the internet with Vercel (frontend) + Render (backend).

1. Push to GitHub (step 2️⃣ above)
2. Deploy frontend to Vercel
3. Deploy backend to Render
4. Wire them together

**Guide:** [DEPLOYMENT.md](./DEPLOYMENT.md)

## 💡 What Makes This Project Special

### 🎨 Design Philosophy
**Minimal Brutalism with Warm Accents** — This isn't a generic template. Every design choice serves a purpose:
- Warm cream background with terracotta accents
- Monospace headers for technical credibility
- Intentional whitespace and asymmetric layouts
- Micro-interactions and thoughtful copy

### 🏗️ Architecture
- **Monorepo structure** — Frontend and backend in one repo
- **Type-safe** — Full TypeScript throughout
- **Scalable** — Clean separation of concerns
- **Deployable** — Production-ready code

### 📊 Features
- **URL shortening** — Generate unique 6-character codes
- **Click tracking** — Log every click with timestamp
- **Analytics** — Beautiful charts showing click trends
- **Dashboard** — Manage all links in one place
- **Link details** — Deep dive into individual link performance
